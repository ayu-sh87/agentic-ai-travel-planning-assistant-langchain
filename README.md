# TravelGenie AI

## Agentic AI-Based Travel Planning Assistant

TravelGenie AI is an intelligent travel planning assistant that helps users:

- Search flights
- Recommend hotels
- Find tourist attractions
- Check weather forecasts
- Estimate trip budget
- Generate AI-based itineraries

---

## Technologies Used

- Python
- LangChain
- Streamlit
- JSON datasets
- Open-Meteo API

# ✈️ TravelGenie AI

## 🌍 Agentic AI-Based Travel Planning Assistant

TravelGenie AI is an intelligent AI-powered travel planning assistant built using **LangChain** and **Streamlit**.  
It helps users generate smart travel plans with flight suggestions, hotel recommendations, weather forecasts, tourist attractions, and estimated budgets — all in one place.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ayu-sh87/agentic-ai-travel-planning-assistant-langchain.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd agentic-ai-travel-planning-assistant-langchain
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Application

```bash
streamlit run app.py
```
